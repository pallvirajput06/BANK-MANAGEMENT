# BANK-MANAGEMENT
A simple Bank Management System built using Python and JSON file storage. It allows users to create accounts, deposit money, withdraw money, update account details, view account information, and delete accounts.
Features:
✔ Create Account
✔ Deposit Money
✔ Withdraw Money
✔ View Account Details
✔ Update User Details
✔ Delete Account
✔ JSON Database Storage

Technologies Used:
- Python
- JSON
- OOP (Object-Oriented Programming)
